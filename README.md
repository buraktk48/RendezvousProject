A simple command-line application that collects identification and personal information from four individuals, allows each person to select an appointment time, and displays all selected time slots at the end of the process. Additionally, the application requests a service type (Visa / Passport Loss) from the user and displays the corresponding informational message.

Sequential data entry for four individuals:

- Turkish ID number, first name, last name, mother’s name, father’s name
- Displaying available appointment time slots (between 09:00 and 09:45)
- Allowing the user to select and store an appointment time
- 
Service type selection:

- V → Visa application information

- P → Passport loss information

At the end of the process:

- Displaying each individual’s personal information and assigned appointment time

- Listing all appointment times selected by the four individuals collectively

Basit bir komut satırı uygulamasıyla 4 kişinin kimlik ve kişisel bilgilerini alır, her kişi için randevu saati seçtirir ve seçilen saatleri işlem sonunda listeler. Ayrıca kullanıcıdan hizmet tipi alınarak (Vize / Pasaport Kaybı) bilgilendirme mesajı gösterir.

4 kişi için sırasıyla bilgi girişi:

- T.C. kimlik no, isim, soyisim, anne adı, baba adı

- Uygun randevu saatlerini listeleme (09.00–09.45 arası)

- Kullanıcının randevu saati seçmesi ve kaydedilmesi

Hizmet tipi seçimi:

- V → Vize başvurusu bilgilendirmesi

- P → Pasaport kaybı bilgilendirmesi

İşlem bitiminde:

- Kişi bilgilerini ve randevu saatini ekrana yazdırma

- 4 kişinin seçtiği randevu saatlerini topluca listeleme
